# Beta invite kit

Copy and personalize this message when inviting someone to the Belief Architecture private beta.

---

**Subject:** You're invited — Belief Architecture beta

Hi [Name],

I'm inviting you to try **Belief Architecture** — a private beta app for reading Scripture, journaling, and mapping your faith framework with AI that stays grounded in *your* journey.

**Your link:** [YOUR_PRODUCTION_URL]

**What to try first (pick one):**
1. **Reader** — open the Bible, highlight a verse, try verse AI
2. **Journal** — write one entry (today's prompt or free-form)
3. **My AI** — ask one question about something you're processing

There's a lot more (Framework, habits, partner mode), but those three loops are the best starting point.

**Install as an app (recommended on phone):**
- **iPhone:** Safari → Share → **Add to Home Screen**
- **Android:** Chrome menu → **Install app** or **Add to Home screen**
- **Desktop:** look for the install icon in the address bar (Chrome/Edge)

**Feedback:** Reply to this email or write to support@beliefarchitecture.app — bugs, confusion, and "this felt meaningful" are all useful.

**Note:** This is beta software. Export journal entries from Settings if you write anything you care about keeping.

Thanks for walking with me on this.

[Your name]
